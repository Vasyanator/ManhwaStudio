# **Installation and launch**

**Note:** the screenshots are captured with the Russian interface. Retaking them in English is a task waiting for a volunteer — pull requests are welcome.

- Go to the releases page at https://github.com/Vasyanator/ManhwaStudio/releases/latest
- Download `manhwastudio_rs.exe (Windows)`/`manhwastudio_rs (Linux)` and run it
  - **! Make sure the path contains no Cyrillic (Russian letters) and no spaces !**
- It is best to install for all users, but if there is little space on drive C, you can pick a custom location on another drive.
- You can choose the basic installation if you just want to explore the program. Choose the full one if you need all the AI models, but that requires downloading about 2 GB more.
- Wait until everything is downloaded
- Open `New project`
- Follow the instructions in the step-by-step mode
