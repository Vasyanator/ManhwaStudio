# **Installation et lancement**

**Remarque :** les captures d'écran sont prises avec l'interface en russe. Les refaire en français est une tâche qui attend une personne volontaire — les pull requests sont les bienvenues.

- Rendez-vous sur la page des releases via ce lien https://github.com/Vasyanator/ManhwaStudio/releases/latest
- Téléchargez le fichier `manhwastudio_rs.exe (Windows)`/`manhwastudio_rs (Linux)` et lancez-le
  - **! Assurez-vous que le chemin ne contient ni caractères cyrilliques (lettres russes) ni espaces !**
- Il est préférable d'installer pour tous les utilisateurs, mais s'il reste peu de place sur le disque C, vous pouvez choisir un emplacement personnalisé sur un autre disque.
- Vous pouvez choisir l'installation de base pour simplement découvrir le programme. Choisissez l'installation complète s'il vous faut tous les modèles d'IA, mais il faudra alors télécharger environ 2 Go supplémentaires.
- Attendez la fin de tous les téléchargements
- Ouvrez `Nouveau Projet`
- Suivez les instructions en mode pas à pas
