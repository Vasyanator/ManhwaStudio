# **Onglet Personnages**

**Remarque :** les captures d'écran sont prises avec l'interface en russe. Les refaire en français est une tâche qui attend une personne volontaire — les pull requests sont les bienvenues.

![image](../images/Вкладка-Персонажи/1.png)

Liste des personnages de la série avec images, nom, description et groupes.
- Les données des personnages (hors images) sont converties en prompt pour la traduction.
- La recherche fonctionne sur tous les paramètres à la fois
- On peut afficher un groupe donné

## **Fenêtre d'édition / de nouveau personnage**
![image](../images/Вкладка-Персонажи/2.png)

- On peut coller une image depuis le presse-papiers
- Définir un nom et une description
- Ajouter le personnage à des groupes existants ou créer un nouveau groupe
