# **Instalação e execução**

**Observação:** as capturas de tela foram feitas com a interface em russo. Refazê-las em português é uma tarefa à espera de alguém voluntário — pull requests são bem-vindos.

- Acesse os releases pelo link https://github.com/Vasyanator/ManhwaStudio/releases/latest
- Baixe o arquivo `manhwastudio_rs.exe (Windows)`/`manhwastudio_rs (Linux)` e execute-o
  - **! Certifique-se de que o caminho não contenha cirílico (letras russas) nem espaços !**
- É preferível instalar para todos os usuários, mas se houver pouco espaço no disco C, você pode escolher um local personalizado em outro disco.
- Você pode escolher a instalação básica apenas para conhecer o programa. Escolha a completa se precisar de todos os modelos de IA, mas nesse caso é necessário baixar cerca de 2 GB adicionais.
- Aguarde até que tudo seja baixado
- Abra `Novo Projeto`
- Siga as instruções do modo passo a passo
