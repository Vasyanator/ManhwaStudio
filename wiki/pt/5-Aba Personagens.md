# **Aba Personagens**

**Observação:** as capturas de tela foram feitas com a interface em russo. Refazê-las em português é uma tarefa à espera de alguém voluntário — pull requests são bem-vindos.

![image](../images/Вкладка-Персонажи/1.png)

Lista dos personagens da série com imagens, nome, descrição e grupos.
- Os dados dos personagens (exceto as imagens) são convertidos em um prompt para a tradução.
- A busca funciona em todos os parâmetros ao mesmo tempo
- É possível exibir um grupo específico

## **Janela de edição/novo personagem**
![image](../images/Вкладка-Персонажи/2.png)

- É possível colar uma imagem da área de transferência
- Definir o nome e a descrição
- Adicionar o personagem a grupos existentes ou criar um novo grupo
