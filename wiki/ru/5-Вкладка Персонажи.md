# **Вкладка Персонажи**
![image](../images/Вкладка-Персонажи/1.png)

Список персонажей тайтла с картинками, именем, описанием и группами.
- Данные персонажей (кроме картинок) конвертируется в промпт для перевода.
- Поиск работает по всем параметрам сразу
- Можно отобразить определенную группу

## **Окно редактирования/нового персонажа**
![image](../images/Вкладка-Персонажи/2.png)

- Можно вставить картинку из буфера обмена
- Задать имя и описание
- Добавить персонажа в существующие группы или создать новую группу
